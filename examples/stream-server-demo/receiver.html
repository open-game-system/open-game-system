🎥 Stream Kit Video Receiver

Connect to a Puppeteer-launched stream to receive video

How to use:

  1. Start the container server: cd container && bun run src/server.ts
  2. Enter a URL below that you want to stream
  3. Click Start Stream & Listen to begin!
  4. The page will automatically start the stream and connect you
  5. You'll see the live content from the Puppeteer browser!
